사용 방법

  1. 1. 아래의 바코드 텍스트를 복사하세요
  2. 2. PDA 입고 페이지에서 컨테이너를 선택하세요
  3. 3. 바코드 스캔창에 텍스트를 붙여넣기하세요
  4. 4. 자동으로 파트가 인식되고 수량이 업데이트되는지 확인하세요

테스트 시나리오

  1. 1. 각 파트별로 여러 번 스캔해서 수량이 누적되는지 확인
  2. 2. 동일한 고유코드로 중복 스캔 시 오류 메시지 확인
  3. 3. 모든 파트 스캔 완료 시 입고 확정 버튼 활성화 확인
  4. 4. 입고 확정 후 실제 재고에 반영되는지 확인
